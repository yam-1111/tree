# Tree Visualizer

An interactive folder visualizer that renders plain text into structured directory trees.

## Features

* **Text-to-Tree Generation:** Build visual directory structures using indentation.
* **Strict Nesting:** Assign child levels using exactly 4 spaces or 1 tab.
* **Automatic Typing:** Suffix a name with a colon (`:`) to render it as a directory.
* **Symbol Preservation:** Wrap filenames in quotes (`'`, `"`, or `` ` ``) to use special characters like colons safely.
* **Comment Blocks:** Enclose instructions or prompts in `"""` to exclude them from the visual output.
